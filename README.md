Full-stack pet products marketplace built with modern JavaScript technologies.

Designed as a scalable e-commerce application featuring catalog browsing, shopping cart logic, product management, backend API integration, and database operations.

---

## 🚀 Live Demo

⚡ Deployment in progress via Railway

Project can be run locally:

```bash
npm install
npm run dev
```

## 📌 Features

- Product catalog with categories
- Product detail pages
- Shopping cart functionality
- REST API integration
- Responsive UI
- State management
- Backend CRUD operations
- SQLite database with Sequelize ORM

## ⚙️ Installation

Clone repository:

```bash
git clone https://github.com/LarysaSperling/pet-shop.git
cd pet-shop
```

Install dependencies:

```bash
npm install
```

Run frontend:

```bash
npm run dev
```

Run backend:

```bash
npm run start
```

---

## API Example

```http
GET /api/products
POST /api/cart
PUT /api/products/:id
DELETE /api/products/:id
```
